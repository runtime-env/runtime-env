# CLI reference

## Top-level options

- `--global-variable-name <name>` (default: `runtimeEnv`)
- `--schema-file <path>` (default: `.runtimeenvschema.json`)
- `--watch`

## Commands

### `gen-js`

- `--env-file <envFile...>`
- `--output-file <outputFile>`

### `gen-ts`

- `--output-file <outputFile>`

### `interpolate`

- `--env-file <envFile...>`
- `--input-file <inputFile>`
- `--output-file <outputFile>`
